02 - Property Binding

constructors are methods executed at the point in time the component is created by Angular

[brackets] indicate property binding, that we want to dynamically bind some property to and expression on the other side of the =

So [property] = "expression", in this case the button property disabled is bound to the expression NOT allowButtonPush, which setsTimeout of two seconds, before expression changes from true to false, which allows the button to be pressed.

Note: The only action that will occur is false will turn to true and the button will change from disabled to !disabled. This occurs after 2 seconds on page load, so there is no interaction after that, the button just becomes clickable

Showing allowButtonPush value using string interpolation: {{ allowButtonPush }}

Showing allowButtonPush value using property binding: 

  [property]="variable with expression or simply an expression sans variable reference"
